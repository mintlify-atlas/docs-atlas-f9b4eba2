---
title: 'Sky: Physically-Based Atmospheric Scattering Sky Shader'
description: 'Adds a physically-based sky to your scene using Three.js Sky shader. Control sun position, turbidity, Rayleigh scattering, and Mie scattering.'
---

`Sky` adds a physically-based atmospheric sky to your scene using the [Three.js Sky shader](https://threejs.org/examples/#webgl_shaders_sky). The sky renders as a large sphere surrounding the scene and reacts to sun position, atmospheric turbidity, and scattering parameters. You can set the sun position directly via `sunPosition` or derive it from `inclination` and `azimuth` angles.

## Props

<ParamField path="distance" type="number" default="1000">
  Radius of the sky sphere. Should be large enough to enclose your entire scene.
</ParamField>

<ParamField path="sunPosition" type="ReactThreeFiber.Vector3">
  Explicit sun position vector. When provided, overrides the position computed from `inclination` and `azimuth`.
</ParamField>

<ParamField path="inclination" type="number" default="0.6">
  Vertical sun angle in the range [0, 1]. `0` is the horizon, `1` is directly overhead. Used to compute `sunPosition` when no explicit vector is given.
</ParamField>

<ParamField path="azimuth" type="number" default="0.1">
  Horizontal sun angle in the range [0, 1]. Used to compute `sunPosition` when no explicit vector is given.
</ParamField>

<ParamField path="mieCoefficient" type="number" default="0.005">
  Mie scattering coefficient. Higher values increase haze and atmospheric glow near the sun.
</ParamField>

<ParamField path="mieDirectionalG" type="number" default="0.8">
  Mie scattering directionality. Values closer to 1 concentrate the glow directly around the sun.
</ParamField>

<ParamField path="rayleigh" type="number" default="0.5">
  Rayleigh scattering coefficient. Controls how blue the sky appears. Higher values produce a deeper blue.
</ParamField>

<ParamField path="turbidity" type="number" default="10">
  Atmospheric turbidity (haziness). Lower values produce a clear sky, higher values a more overcast look.
</ParamField>

## Usage

### Default sky

Add a sky with default sun position.

```jsx
import { Sky } from '@react-three/drei'

function Scene() {
  return (
    <>
      <Sky />
      <mesh>
        <planeGeometry args={[100, 100]} />
        <meshStandardMaterial color="green" />
      </mesh>
    </>
  )
}
```

### Custom sun position with angles

Drive the sun position via `inclination` and `azimuth`.

```jsx
<Sky
  distance={450000}
  inclination={0.49}
  azimuth={0.25}
  turbidity={8}
  rayleigh={2}
/>
```

### Explicit sun position vector

Pass a `sunPosition` vector directly for precise control.

```jsx
<Sky sunPosition={[0, 1, 0]} distance={450000} />
```

### Sunset atmosphere

Simulate a low-sun, hazy sunset by adjusting inclination and scattering.

```jsx
<Sky
  distance={450000}
  inclination={0.52}
  azimuth={0.25}
  turbidity={10}
  rayleigh={3}
  mieCoefficient={0.005}
  mieDirectionalG={0.7}
/>
```

<Tip>
  Pair `Sky` with a `DirectionalLight` whose position matches `sunPosition` to cast consistent sun shadows across your scene.
</Tip>
